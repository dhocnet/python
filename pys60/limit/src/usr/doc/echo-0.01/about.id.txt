echo simulation v0.01 beta

Perintah tiruan dari perintah echo
yang terdapat pada Unix.

echo simulation pada limit hanya
membawa fungsi dasar. Yaitu menampilkan
string pada layar, menyimpan string
pada sebuah file, menambahkan string
pada file yang telah ada sebelumnya
dan mengganti isi suatu file dengan
string baru.

Informasi untuk program echo pada Unix
silahkan merujuk pada halaman berikut:
http://www.gnu.org/software/coreutils/manual/html_node/echo-invocation.html
http://en.wikipedia.org/wiki/Echo_(command)
