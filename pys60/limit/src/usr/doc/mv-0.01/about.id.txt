mv simulation v0.01 beta

mv simulation digunakan untuk
memindah atau me-rename file
dan folder. Program mv yang
asli dapat ditemukan disetiap
mesin unix dan unix-like.

Informasi untuk perintah mv
pada unix, silahkan merujuk
pada halaman berikut:
http://www.gnu.org/software/coreutils/manual/html_node/mv-invocation.html
