Limit versi 0.01 beta

Limit adalah sebuah program simulasi Unix Terminal sederhana yang di tulis menggunakan bahasa pemrograman Python untuk seri 60.

Tujuan utama pembuatan Limit adalah sebagai alternatif untuk para pengguna symbian versi 1 (satu), dimana pada program lain yang sejenis, yaitu PyUnix, tidak disuport oleh perangkat tersebut (S60 v1).

Limit ditulis hanya sebagai interpreter yang sangat sederhana dimana kamu sebagai pengguna sangat dimungkinkan untuk menuliskan program atau perintah buatanmu sendiri. Tentu saja, program atau perintah yang kamu tulis haruslah menggunakan bahasa pemrograman python untuk s60. Untuk lebih lengkapnya, silahkan lihat file limit-dev-howto.?.txt.
