
Nama : compressed.types

compressed.types adalah file yang berisi daftar ekstensi file kompresan.

File ini di baca (dibutuhkan) oleh program ls untuk mendeteksi setiap file yang dilihatnya.

File compressed.types adalah file yang dapat di rubah isinya menggunakan editor teks biasa karena file ini adalah file teks murni.

Perlu diingat! Saat menyimpan hasil perubahan, file harus disimpan dalam format teks murni (ansi, bukan unicode) tanpa di sisipi oleh format koding tertentu atau file tidak akan dapat dieksekusi.

File compressed.types terletak di /usr/share/mime
