cp simulation v0.01 beta

Sebuah program tiruan dari
unix-like yang berfungsi untuk
mengkopi file dan direktori.

Pada versi awal, hanya mendukung
perintah dasar saja. yaitu
menyalin dari file satu ke file
lainya.

Informasi untuk perintah cp pada
unix, silahkan merujuk pada
halaman berikut:
http://www.gnu.org/software/coreutils/manual/html_node/cp-invocation.html
http://en.wikipedia.org/wiki/Cp_(Unix)
