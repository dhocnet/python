:::TENTANG PROGRAM:

PassBox adalah sebuah program sederhana yang berfungsi sebagai 
pengingat password yang berjalan pada ponsel-ponsel symbian 
seri 60. PassBox ditulis menggunakan bahasa pemrograman (scripting) 
Python For S60.

Tujuan pembuatan passbox pada awalnya hanya untuk mempermudah 'saya'
mengingat akun-akun yang saya miliki di beberapa layanan online, 
hingga suatu saat ada keinginan untuk membagikanya kepada umum.
Siapa tau saja program kecil ini bisa berguna bagi yang lain :D
Ya, dan siapa yang tau ?

Bagi yang ingin berpartisipasi untuk mengembangkan/ memperbaiki
program ini, saya sangat senang dan mempersilahkan sekali. Karena 
dengan begitu kita bisa saling berbagi dan tukar pikiran tentang 
ide-ide baru.
=================================================================

:::KEBUTUHAN SISTEM:

Passbox dibuat menggunakan bahasa pemrograman python for s60. Jadi
untuk dapat menggunakanya, Anda harus memiliki interpreter python
for s60 pada sistem ponsel Anda.

Bila Anda belum mempunyai interpreter python for s60 pada sistem
Anda, silahkan unduh dan install python for s60 dari 

http://sourceforge.net/projects/pys60/

Program PassBox hanya pernah diuji coba pada:
- Ponsel Nokia(TM) N-GAGE QD(R) (Symbian seri 60 v61)
- Python module pack v1.33
- Python For S60 v1.45

:::REGISTRASI

Registrasi pada program ini bukanlah registrasi untuk pembelian
perangkat lunak (lisensi), tapi registrasi pembuatan akun Anda
pada aplikasi.

Secara otomatis, saat penggunaan pertamakali (setelah instalasi)
program akan langsung menampilkan konfirmasi untuk pembuatan 
akun baru. Tekan [OK] untuk menyetujuinya dan form registrasi
akan segera ditampilkan.

Isikan data-data yang diminta seperti Username dan Password.
Khusus untuk password, Anda akan diminta untuk memasukanya 
sebanyak 2 (dua) kali untuk memastikan bila tidak ada salah
ketik dalam penulisan password.

CATATAN:
    Setelah proses registrasi Anda lalui, password sudah
    tidak dapat lagi dilihat. Jadi, jangan sampai lupa
    dengan password yang satu ini karena program ini tidak 
    memiliki fasilitas "Reset Password" atau "Password Recovery"
    
    Untuk username, sebaiknya Anda memilih yang umum saja
    karena setelah registrasi berhasil, username tidak dapat
    diganti lagi.
